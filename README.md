# Roboflow Supervision Computer Vision Experiments

This repository contains a Colab notebook for experimenting with a supervised computer vision workflow using Roboflow, Ultralytics YOLOv8, OpenCV, and the `supervision` annotation library.

The current experiment trains a YOLOv8 object detection model on a Roboflow-exported truck detection dataset, runs inference on test images, and visualizes detections with bounding boxes, class names, and confidence scores.
